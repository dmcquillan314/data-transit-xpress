userPreference
directive in module userPreference

Description

This directive injects update and retrieval operations for user preferences

Usage

as attribute
<ANY user-preference
     preference-key="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
preferenceKeystring

The name of the preference that this directive instance will control

Properties

  • currentPreference

    the object in the scope representing the current value of the preference

  • updatePreference(newPreference)

    a function to update the preference value for this preference
