ppdProductText
directive in module product

Description

This directive, given a product text object will inject that object into a template to display a common markup

Usage

as element:
<ppd-product-text
       extended-info="{object}">
</ppd-product-text>
as attribute
<ANY ppd-product-text
     extended-info="{object}">
   ...
</ANY>

Parameters

ParamTypeDetails
extendedInfoobject

a product text object from a returned product
