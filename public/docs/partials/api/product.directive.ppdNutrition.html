ppdNutrition
directive in module product

Description

This directive, given a nutrition object will inject that object into a template to display a common markup

Usage

as element:
<ppd-nutrition
       nutrition="{object}">
</ppd-nutrition>
as attribute
<ANY ppd-nutrition
     nutrition="{object}">
   ...
</ANY>

Parameters

ParamTypeDetails
nutritionobject

a nutrition object from a returned product
