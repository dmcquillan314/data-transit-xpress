CategoryFilter
filter in module product

Description

This is a filter to take in a normalized set of products and convert it to a set, keyed by category id, with the properties items(the set of objects in that set) and catName(the descriptive category name for that set).

Usage

In HTML Template Binding

{{ CategoryFilter_expression | CategoryFilter:shouldTransform }}

In JavaScript

$filter('CategoryFilter')(data, shouldTransform)

Parameters

ParamTypeDetails
dataarray

Collection of product objects. Each object must have a value for the consumCatId property

shouldTransformboolean

Boolean parameter that will be evaluated to see if the data should be transformed

(default: true)
