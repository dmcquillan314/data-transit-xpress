    object used to store the properties for the guest user form

    used to post the guest user object to the Peapod rest services

  • submitGuestUser($event,guestUserForm)

    method to handle the callback from the form if the user hits enter or 'go' on a mobile device. This method calls sendGuestUser as a delegate

  • sendUser(user,userForm)

    used to post the user object to the Peapod rest services

  • submitUser($event,userForm)

    method to handle the callback from the form if the user hits enter or 'go' on a mobile device. This method calls sendUser as a delegate.
