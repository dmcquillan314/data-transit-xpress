CartStatusMessageService
service in module cart

Description

This service manages status messages from the response of adding an item to a cart

Properties

  • cartStatusMessages

    a managed object to cache all error messages for cart addition

  • clearStatusMessageForProdId(prodId)

    clears the error message for a prod id from the cartStatusMessages cache

  • lookupStatusMessageForProdId(prodId)

    lookup the associated error message for a prodId

  • setStatusMessages(resultArray)

    given a response from adding/removing an item to/from the cart it will preprocess those messages to instantiate or add to the cartStatusMessages cache
