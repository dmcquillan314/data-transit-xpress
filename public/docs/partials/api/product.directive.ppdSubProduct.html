ppdSubProduct
directive in module product

Description

This directive injects SubstituteProductController into this element.

Usage

as element:
<ppd-sub-product
       product="{object}"
       product-id="{number}">
</ppd-sub-product>
as attribute
<ANY ppd-sub-product
     product="{object}"
     product-id="{number}">
   ...
</ANY>

Parameters

ParamTypeDetails
productobject

injects a product object into the isolated scope to be used for display

productIdnumber

used to pass a productId to the controller
