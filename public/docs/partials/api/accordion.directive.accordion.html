accordion
directive in module accordion

Description

This directive simply sets up the directive controller and adds an accordion CSS class to the element itself.

Usage

as element:
<accordion
       [close-others="{boolean}"]>
</accordion>
as attribute
<ANY accordion
     [close-others="{boolean}"]>
   ...
</ANY>

Directive info

  • This directive executes at priority level default.

Parameters

ParamTypeDetails
closeOthers
(optional)
boolean

This tells the controller whether it should close all active groups before opening the next.

Example

Title content 

    Content of accordion

</accordion-group>
<accordion-group>
    <accordion-heading>
    Title content 2
    </accordion-heading>

    Content of accordion 2

</accordion-group>

    Content of accordion

</accordion-group>
<accordion-group heading="title content 2">

    Content of accordion 2

</accordion-group>
