ppdAddToCart
directive in module cart

Description

used with AddToCartController to provide the add to cart button

Usage

as element:
<ppd-add-to-cart
       product="{object}"
       [trash-button="{boolean}"]
       [show-subtotal-tooltip="{boolean}"]>
</ppd-add-to-cart>
as attribute
<ANY ppd-add-to-cart
     product="{object}"
     [trash-button="{boolean}"]
     [show-subtotal-tooltip="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
productobject

provides the product to be used with the button

trashButton
(optional)
boolean

shows the trash button to remove the item from the cart

(default: false)

showSubtotalTooltip
(optional)
boolean

shows a tooltip with the current subtotal when an interaction is made with the button

(default: false)
