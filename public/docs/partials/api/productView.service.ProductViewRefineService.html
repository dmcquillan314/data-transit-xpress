ProductViewRefineService
service in module productView

Description

This service manages a hash to be used with the product search facets.

Properties

  • facets

    This is the set of facets that were returned from the injected provider

  • isFiltersProcessing

    This flag should be set to true whenever an update is processing.

  • destroy

    This method resets the state of the service

  • resetFiltersByType(type)

    Will reset the stored type property of filters

  • resetFilters()

    Resets the value of the filters to {}

  • toggleFilter(type,id)

    Will toggle the existence of the id in the array stored in the type property of filters

  • hasFilter(type,id)

    Returns if a type and id is present in the stored filters

  • getFilters()

    Returns the stored filters object

  • getFiltersByType(type)

    Returns the stored type array from the filters object.
