keyedRouteService
service in module keyedRoute

Properties

  • getKeyedArray(params)

    Description:

    Will take a string in the form of a url and produce a map of arrays.

    Parameters:

    • params - a url string

    Example:
        keyedRouteService.getKeyedArray( '/categories/0/categories/1/categories/2/products/1' );

    {
        'categories': [ 0,1,2 ],
        'products': [1]
    }
