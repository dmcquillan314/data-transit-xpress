CertonaRecommendationsService
service in module recommendations

Description

This factory provides a callback promise to be used for recommendations. The options parameters can take a set of options to be passed to certona. There are two exposed methods from this service that both take the same set of parameters.

  1. sendData - simply sends the request data to certona and should timeout without a response.
  2. retrieveRecommendations - sends the request data to certona and resolves a promise with the recommendations data.

Usage

CertonaRecommendationsService([top1][, top2][, rrelem][, rrnum][, event][, itemid][, exitemid][, timeout][, options]);
