tabset
directive in module ui.bootstrap.tabs

Description

Tabset is the outer container for the tabs directive

Usage

as element:
<tabset
       [vertical="{boolean}"]
       [justified="{boolean}"]>
</tabset>
as attribute
<ANY tabset
     [vertical="{boolean}"]
     [justified="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
vertical
(optional)
boolean

Whether or not to use vertical styling for the tabs.

justified
(optional)
boolean

Whether or not to use justified styling for the tabs.
