animateHeight
directive in module animate

Description

This directive will animate the height of the directive given the height of its contents. If no additional height is specified then it will calculate the height of the element and add that value to the height of the element via the style attribute of the element.

Usage

as attribute
<ANY animate-height
     [additional-height="{number}"]
     condition="{boolean}">
   ...
</ANY>

Directive info

  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
additionalHeight
(optional)
number

Additional height to add to the element default is 0

conditionboolean

will remove this element from dom if false, add and animate height if true. Will emit the events: loading.directive#refreshScroller and reloadHeights
