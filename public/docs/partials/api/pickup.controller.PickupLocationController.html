PickupLocationController
controller in module pickup

Description

This controller is used for retrieving a set pickup locations and displaying it on a map.

Properties

  • pickupLocations

    This is a collection of pickup locations

  • radii

    An array of radii to be used as a default.

  • pupErrorState

    This is a flag that is set to true when a promise fails

  • errorCode

    This is the associated code with the pupErrorState

  • staticMapUrlForLocation(location,width)

    Retrieves a url for a map with a specified width and location

  • directionsUrlForLocation(location)

    Retrieves a url to direct a user to a google maps url to view it there.

  • lookupLocations(zip,radius)

    Refreshes the pickupLocations for a given zip and radius.

  • lookupLocationsNearMe(radius)

    Using the zip from geolocation will delegate to lookupLocations(zip,radius)

  • iconImageForIndexAndState(index,state)

    Using the index of the pickupLocation and the current state of it which has the following possible values: ['','selected','hover']. This is used to retrieve a different image depending on the state.
