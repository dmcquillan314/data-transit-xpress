ProductViewRefineController
controller in module productView

Description

This controller is used to monitor user interactions and the state of the product result refine overlay

Properties

  • hasFilter

    used to check if a filter is active

  • hasFacet

    used to check if the backing service contains a given facet

  • resetFilters

    used to clear all selected filters

  • resetFiltersByType

    • used to clear selected filters by facet name

  • toggleFilter(type,filterId)

    • used to select/deselect a filter

  • facets

    current set of facets available to the user to choose from
