FormHelperService
service in module form

Description

Contains a set of methods to be injected into the scope of a controller using the injectFormHelperMethodsToScope method.

Properties

  • injectFormHelperMethodsToScope($scope)

    given a scope it will inject all the helper methods to that scope

  • isControllerInErrorState(ngModelController)

    [helperMethod] checks if a controller is in an error state.

  • getCssClasses(ngModelController)

    [helperMethod] used to add/remove error state classes from the form elements within all login forms

  • showError(ngModelController,error)

    [helperMethod] used to get an error message for a specified FormFieldController
