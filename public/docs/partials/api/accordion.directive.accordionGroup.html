accordionGroup
directive in module accordion

Usage

as element:
<accordion-group
       [is-open="{boolean}"]
       [heading="{string}"]>
</accordion-group>
as attribute
<ANY accordion-group
     [is-open="{boolean}"]
     [heading="{string}"]>
   ...
</ANY>

Directive info

  • This directive executes at priority level default.

Parameters

ParamTypeDetails
isOpen
(optional)
boolean

Condition for when this group should be open

heading
(optional)
string

Text for header if no dom structure is required. For more complex header markup or dynamic text resolution use accordionHeader
