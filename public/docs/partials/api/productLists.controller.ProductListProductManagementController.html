ProductListProductManagementController
controller in module productLists

Description

A controller used to perform operations on a given product list. Handles addition and removal of products from a list as well as retrieval of the lists that a product belongs to.

Properties

  • getClassIfProductInList(list)

    should be used with the ng-class directive.

  • toggleProductInList(list)

    used to toggle a product in a list

  • removeProductFromListAndRefresh(list,productId)

    remove a given product from a given list and refresh the products in that list
