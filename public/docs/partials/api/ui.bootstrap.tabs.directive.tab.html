tab
directive in module ui.bootstrap.tabs

Description

Creates a tab with a heading and content. Must be placed within a tabset.

Usage

as element:
<tab
       [heading="{string}"]
       [select="{string}"]
       [active="{boolean}"]
       [disabled]>
</tab>
as attribute
<ANY tab
     [heading="{string}"]
     [select="{string}"]
     [active="{boolean}"]
     [disabled]>
   ...
</ANY>

Parameters

ParamTypeDetails
heading
(optional)
string

The visible heading, or title, of the tab. Set HTML headings with tabHeading.

select
(optional)
string

An expression to evaluate when the tab is selected.

active
(optional)
boolean

A binding, telling whether or not this tab is selected.

disabled
(optional)
boolean

A binding, telling whether or not this tab is disabled.
