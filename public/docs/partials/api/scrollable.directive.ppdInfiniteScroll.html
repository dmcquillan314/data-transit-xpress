ppdInfiniteScroll
directive in module scrollable

Description

This directives creates a place to put a listener of the current scroll position. When the threshold is exceeded it will trigger the callback expression in the ppdInfiniteScroll param.

Usage

as attribute
<ANY ppd-infinite-scroll
     [threshold="{number}"]
     [ppd-infinite-scroll]="{function|expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
threshold
(optional)
number

Sets the scroll threshold from the bottom of the scroll pane from which to call the update handler

(default: 0)

[ppdInfiniteScroll]functionexpression

sets a callback expression to be ran when the threshold has been exceeded
