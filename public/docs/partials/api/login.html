This module contains all relevant code to guest and user login

Example

  • User submits form with zip code -> LoginController.user.zip = 60089
  • Trigger event listen for key id 13 -> submitUser($event, userForm)
  • Calls sendUser from submitUser
  • Calls sendGuestUser
  • Response is returned from rest api which calls handleGuestError
  • Cities property is injected into the scope
  • User selects a city and then will repeat steps 2 - 4
  • Response is returned from rest api which calls handleGuestSuccess
  • User is logged in
