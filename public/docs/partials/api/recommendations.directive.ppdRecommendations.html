ppdRecommendations
directive in module recommendations

Description

This factory provides a callback promise to be used for recommendations. The options parameters can take a set of options to be passed to certona.

Usage

as attribute
<ANY ppd-recommendations
     [top1="{number}"]
     [top2="{number}"]
     [rrelem="{string}"]
     [rrnum="{string}"]
     [event="{string}"]
     [itemid="{string=|array}"]
     [exitemid="{string=|array}"]
     [timeout="{number}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
top1
(optional)
number

Not sure

(default: 100000)

top2
(optional)
number

Not sure

(default: 100000)

rrelem
(optional)
string

Recommendations type for certona

rrnum
(optional)
string

number of recommendations

(default: 3)

event
(optional)
string

event for certona recommendations

itemid
(optional)
string=array

A semicolon delimited list of item SKUs to be used to do content based recommendations. If it is an array then it will be joined with ';'

(default: '')

exitemid
(optional)
string=array

A semicolon delimited list of item SKUs to be excluded from the results. If it is an array then it will be joined with ';'

(default: '')

timeout
(optional)
number

Timeout for the backing promise of the request to certona. (This will trigger the call to cancel itself)

(default: 4000)
