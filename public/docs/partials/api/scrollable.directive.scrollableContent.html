scrollableContent
directive in module scrollable

Description

This directive adds a scroller trait to this dom element. It will also support legacy devices by adding the overthrow polyfill to this element.

This directive is overloaded and will broadcast the current scroll position.

This directive accepts the following events:
scrollToElement - This will scroll to an element specified in the data parameters (e.g. { element: }.

This directive broadcasts the following events:
scrollPositionUpdated - This broadcasts the current scroll position (e.g. { currentPosition: currentScrollPosition, maxScrollPosition: maxScrollPosition } )

Usage

as attribute
<ANY scrollable-content>
   ...
</ANY>
