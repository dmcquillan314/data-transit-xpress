ForgotPasswordService
service in module forgot

Description

This service contains functions to handle various error/success cases for password

Properties

  • handleForgotPasswordSuccess(response)

    Description:

    this method reads the response from the Peapod rest api and then sends them to the successful forgot page.

    Parameters:

    • response - response from forgot username api service

  • handleForgotPasswordError(user,forgotPasswordForm,response)

    Description:

    this method reads the response from the Peapod rest api and then sends them to the successful forgot page.

    Parameters:

    • user - representative object for the data sent to the rest api
    • forgotUsernameForm - the injected form scope object for forgot password
    • response - response from guest login api service
