AddToCartController
controller in module cart

Description

Handles operations to do with the add to cart button.

Properties

  • decrementQuantity(qty)

    used to decrement the quantity of a given product in the cart

  • incrementQuantity(qty)

    used to increment the quantity of a given product in the cart

  • addToCart()

    used to add an item to the cart

  • closeDialog()

    used to dismiss a product warning or restriction message

  • removeFromCart()

    used to remove a product from the cart
