ProductViewSubheaderController
controller in module productView

Description

This controller is used to handle user interactions in the header of the product view page

Properties

  • toggleViewType

    • used to toggle the view type stored in the ProductViewSubheaderService

  • openRefineView

    • calls the following method call -> $scope.openNamedModalView( 'modal-right', 'refine' );

  • toggleEditMode

    • used to toggle the edit mode for the product display

  • viewType

    (this variable is watched for changes) - ['grid'|'list']

  • headerText

    (this variable is watched for changes) - object from the config object within the data object

  • showMobileBackButton

    (this variable is watched for changes) - used with the ProductViewTreeController

  • showEditButton

    (this variable is watched for changes) - used to show or hide the edit button depending on the configuration of the provided service

  • headerButtonActive

    (this variable is watched for changes) - used to provide whether or not the edit button is active

  • sortTypes

    • used to provide the sort types the user can choose from (this can be overridden from the provider config)
