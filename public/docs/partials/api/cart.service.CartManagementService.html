CartManagementService
service in module cart

Description

This is a service to manage the items in the cart

Properties

  • storedDataUpdated

    a boolean flag to tell other components that are watching it that the cart has been updated. This is a convenience property to avoid having to watch the entire cart cache, which would be an expensive operation

  • setStoredCartData

    a setter for storedCartData

  • getStoredCartData

    a getter for storedCartData

  • retrieveTransformedStoredCartData

    retrieve a product set categorized by consumCatId.

  • lookupProductData(prodId)

    will lookup if the cached cart object has this product
