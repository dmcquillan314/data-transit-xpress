ppdProductListProductManagement
directive in module productLists

Description

will inject the ProductListProductManagementController into the this location to provide necessary functionality to manage a set of products in a product lists. This directive is adaptive based on the value of retrieve lists and will use the following parameters in each case

retrieve-lists == true

  • productId
  • productCollection

retrieve-lists != true

  • currentList

Usage

as element:
<ppd-product-list-product-management
       [product-id]="{number}"
       [retrieve-lists="{boolean}"]
       [[product-collection]="{array}"]
       [current-list="{object}"]>
</ppd-product-list-product-management>
as attribute
<ANY ppd-product-list-product-management
     [product-id]="{number}"
     [retrieve-lists="{boolean}"]
     [[product-collection]="{array}"]
     [current-list="{object}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
[product-id]number

product id to lookup the product lists for

retrieve-lists
(optional)
boolean

whether or not the product lists for this product should be resolved

(default: true)

[product-collection]
(optional)
array

collection of products currently in a list

current-list
(optional)
object

object representing the current list

(default: {})
