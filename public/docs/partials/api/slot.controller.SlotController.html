SlotController
controller in module slot

Description

This controller manages a user's interaction with the delivery/pickup slot flows. It also handles delegation of a user to pickup locations when necessary

Properties

  • selectPickupLocation(location)

    Delegates the user to the pickup location flow

  • selectServiceType(type)

    Selects a service type of either 'P' or 'D' and gets a default set of slots

  • toggleServiceType(type)

    Takes the user's currently selected service type then chooses the other and delegates to selectServiceType

  • reserveSlot(slot)

    Reserves a given slot

  • selectHeader(header)

    Will change the active header and retrieve slots for it

  • resetRestrictedSlot()

    Resets the actively visible restricted items

  • setActiveRestrictedSlot(slot)

    Selects which restricted slot will be visible

  • toggleRestrictedSlot(slot)

    Toggles a restricted slot from open to closed and vice versa

  • removeRestrictedItemsForSlot(slot)

    Removes the restricted items from the cart

  • getAggregatedRestrictedItems(slots)

    Delegates to the aggregatedRestrictedItems method of slot.service:RestrictedSlotService

  • isHeaderSelected(header,viewDate)

    Checks to see if the selected viewDate matches the viewDate on the header

  • isHeaderAvailable(header)

    Checks if the status code of a header matches a set of status codes that should be seen as unavailable

  • getDayClassesForHeader(header,viewDate)

    Retrieves the set of classes that should be added/removed to the element(used with ng-class directive)

  • isSlotRestricted(slot)

    Checks if a slot is restricted

  • restrictedSlotsExist(slot)

    Utility function to tell if a slot has restricted items

  • getRestrictedTypes(slots)

    Delegates to the getRestrictedTypesFromSlots method of slot.service:RestrictedSlotService

  • getRestrictedTypesString(slots)

    Given a set of slots it will generate a common list of restrictions for the given header

  • getSlotClasses(slot,isLast)

    Gets the classes for a given slot and the $last parameter from ng-repeat

  • hasMessage(slot)

    Checks if the slot has a message to display
